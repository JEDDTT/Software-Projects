Data Analysis Udemy and Data Science Salary

My project revolves around a comprehensive analysis of data sourced from Udemy, a leading online learning platform, alongside a dataset tailored to the field of data science taken from kaggle. The primary objective is to extract valuable insights and patterns that can inform strategic decision-making for both Udemy and the data science community. I begin by conducting exploratory data analysis on Udemy's dataset, focusing on user behavior and course preferences. This analysis will offer Udemy insights into user engagement, helping them optimize course offerings and marketing strategies. Simultaneously, I delve into the data science dataset to uncover valuable trends, such as demand for specific skills and salary trends within USA, UK and Canada for data science jobs. These findings will serve as a valuable resource for data scientists, job seekers, and educational institutions, aiding in career decisions and curriculum development. Ultimately, My project aims to bridge the gap between online learning and the burgeoning field of data science, offering actionable insights for Udemy's platform enhancements and contributing to the data science community's knowledge base. Through thorough analysis and visualization, I intend to provide meaningful recommendations for stakeholders in both domains, ensuring data-driven success.
